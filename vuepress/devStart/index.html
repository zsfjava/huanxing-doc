# Nacos 启动&配置修改

# 1. nacos 下载 2.3.2 稳定版本

 https://download.nacos.io/nacos-server/nacos-server-2.3.2.zip

# 2. 修改 nacos 项目 conf/application.properties 文件

db.url.0=jdbc:mysql://huanxing-mysql:3306/huanxing_nacos...
db.user.0=root
db.password.0=123456

# 3. 启动 nacos

Linux/Unix/Mac: sh startup.sh -m standalone
Windows: startup.cmd -m standalone

浏览器访问 http://127.0.0.1:8848/nacos 默认账号密码（nacos/nacos）

# 4. 编辑 nacos application-dev.yml 修改 redis 配置

redis:
  host: huanxing-redis
  port: 6379
  password: #没密码留空
  database: 1

# 5. nacos 修改数据库配置

配置文件名称
huanxing-auth-dev.yml
huanxing-user-biz-dev.yml
huanxing-miniapp-biz-dev.yml

huanxing-promotion-biz-dev.yml
